## Repository Structure

### bapi/

* z_bapi_mat_creation.abap
  → Material creation using BAPI

### bdc/

* z_bdc_call_transaction_mat_creation.abap
  → BDC using CALL TRANSACTION
* z_bdc_session_method_mat_creation.abap
  → BDC using Session Method

### classes/

* zcl_material_service.abap
  → Service class for material data retrieval
* zcl_alv_helper.abap
  → Reusable ALV display class using SALV
* zcl_bdc_helper.abap
  → Utility class for BDC dynpro and field handling

### reports/

Includes multiple report types:

* Classical Reports
* Interactive Reports
* ALV Reports (Basic, Block, Double Click, Modify)
* FOR ALL ENTRIES based report
* Production ALV Report

Examples:

* z_classical_rp_basic.abap
* z_alv_interactive_rp.abap
* z_for_all_entries_rp.abap
* z_production_alv_report.abap
